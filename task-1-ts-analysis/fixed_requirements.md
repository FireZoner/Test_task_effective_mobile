1) Пользователь может добавить в корзину от 1 до 10 единиц одного товара.
2) В корзине может находиться не более 5 различных товаров.
3) Суммарное количество всех товаров в корзине не может превышать 20 штук.
4) При попытке добавить товар, превышающий лимиты, товар НЕ добавляется в корзину, состояние корзины остается неизменным. Система показывает соответствующее сообщение:
	- При превышении лимита одного товара (больше 10 единиц): "Вы можете добавить не более 10 единиц этого товара".
	- При превышении лимита разных товаров (больше 5): "В корзине может быть не более 5 разных товаров".
	- При превышении суммарного лимита (больше 20 единиц): "Общее количество товаров в корзине не может превышать 20 штук".


5) Пользователь может изменять кол-во каждого товара при помощи кнопок "+" и "-".
6) При уменьшении количества товара до нуля товар автоматически удаляется из корзины. Также можно удалить товар при помощи отдельной кнопки "Удалить".
7) Пользователь не может установить количество товара меньше 0.

8) Цена на товар фиксируется в момент добавления в корзину и не меняется при изменении цены в каталоге.
9) Если цена на товар изменилась в каталоге после добавления в корзину, система отображает рядом с товаром уведомление: "Цена изменилась. Обновить?" с кнопкой обновления.
10) При нажатии на кнопку обновления цена в корзине заменяется на актуальную из каталога. При отказе цена остается старой.

11) На странице корзины отображается:
	- Список товаров
	- Количество каждого товара
	- Цена за единицу
	- Общая стоимость позиции
	- Общая стоимость всей корзины
	- Блок с рекламой (п. 12-14)

12) В корзине может отображаться реклама других товаров. Реклама размещается в отдельном блоке под списком товаров.
13) Формат рекламы - кликабельный баннер с изображением, названием товара и кнопкой "Перейти".
14) Реклама отображается:
	- В будние дни (понедельник - пятница)
	- Утром (09:00 - 11:00) и вечером (19:00 - 21:00)
	- Время привязывается к часовому поясу по IP пользователя
	- При загрузке или перезагрузке страницы корзины фронтенд отправляет запрос к бэкенду, передавая текущее время пользователя. Бэкенд возвращает список рекламных баннеров (выбирается маркетологами), актуальных для данного временного интервала
	- В течение одной загрузки страницы (без перезагрузки) содержимое рекламного блока не меняется

15) При добавлении товара в корзину система проверяет его на наличие на складе/кухне/пункте выдачи
16) Если товара нет в наличии:
	- Товар НЕ добавляется в корзину
	- Система показывает сообщение: "Товар временно отсутствует. Добавьте позже"

17) Если товар, уже находящийся в корзине, был удален из каталога магазина:
	- Товар остается в корзине
	- Рядом с товаром отображается пометка *"Товар недоступен"*
	- Оформление заказа с таким товаром невозможно
	- Пользователю предлагается удалить товар из корзины
